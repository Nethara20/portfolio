Portfolio

My personal portfolio site — built from scratch with hand-written HTML, CSS, and JavaScript (no framework, no template).

Live site: https://nethara20.github.io/portfolio/


Concept

The design is built around the bisokotuwa — a valve-pit invention from Anuradhapura, Sri Lanka
(my hometown), dating to around 300 CE, that regulated water pressure in ancient reservoirs. The
site's structure mirrors a real tank cascade: each section is labeled by "elevation" instead of a
generic name, a water channel animates down the left edge connecting every section, and valve-gate
dividers mark the transition between them. It's a layout built around a real story, not a template.


Tech stack

HTML5 / CSS3 (custom properties, grid, no framework)
Vanilla JavaScript (IntersectionObserver for scroll reveals, no dependencies)
Google Fonts: Cinzel (display) + Inter (body) + IBM Plex Mono (labels)
